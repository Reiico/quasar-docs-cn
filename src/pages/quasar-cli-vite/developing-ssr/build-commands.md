title: SSR 构建命令

---

## 开发
```bash
$ quasar dev -m ssr

# ..或者更长的格式:
$ quasar dev --mode ssr
```

## 打包生产环境代码
```bash
$ quasar build -m ssr

# ..或者更长的格式:
$ quasar build --mode ssr
```

如果您想打包生产环境代码时带上调试能力：

```bash
$ quasar build -m ssr -d

# ..或者更长的格式:
$ quasar build -m ssr --debug
```
